## 🚀 About Me

I'm an IT professional with nearly **15 years of hands-on experience**, following a self-taught and apprentice-style learning journey.

My career started in the field, installing internet connections, pulling cables, configuring modems, and helping customers get online. Since then, I've worked in corporate IT across the **Retail**, **Healthcare**, and **Insurance** sectors.

Today, I work primarily in **End User Support**, supporting Windows-based enterprise environments and assisting with the day-to-day administration of technologies such as:

- Active Directory
- Group Policy
- Microsoft Endpoint Configuration Manager (SCCM)
- Windows Server
- Microsoft 365
- Endpoint Management

Most of my professional experience has been built around the Microsoft ecosystem.

Over the last year, I've been expanding my knowledge by learning **Linux**, **Open Source Software (FOSS)**, and the fundamentals of **Cloud**, **Automation**, and **DevOps** through hands-on practice.

I don't claim to know everything.

I simply enjoy learning, experimenting, and understanding how things work beneath the surface.
